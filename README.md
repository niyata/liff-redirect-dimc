## Setup

1. Add these environment variables to Netlify:
   - `XATA_API_KEY`
   - `XATA_DB_URL`

2. Your `XATA_DB_URL` will look like:
   `https://<workspace-id>.xata.sh/db/<database-name>:main`

3. Deploy to Netlify

4. Done!
